#writerside ## Minecraft All-in-One Plugin ### Funktionen Dieses Plugin bietet eine Vielzahl von Funktionen, darunter: - Ein Wirtschaftssystem (Economy) - Viele nützliche Befehle - Verschiedene Listener für Events ### Befehle ```writerside commands: - name: test description: Ein einfacher Testbefehl. usage: /test - name: heal description: Heilt den Spieler vollständig. usage: /heal - name: feed description: Füllt die Hungeranzeige des Spielers. usage: /feed - name: fly description: Aktiviert oder deaktiviert das Fliegen für den Spieler. usage: /fly - name: boom description: Erzeugt eine Explosion um den Spieler. usage: /boom - name: balance description: Zeigt den aktuellen Kontostand des Spielers an. usage: /balance - name: pay description: Sendet Coins an einen anderen Spieler. usage: /pay <Spieler> <Betrag> - name: shop description: Ermöglicht den Kauf eines Items mit der Ingame-Währung. usage: /shop <item> <price>
